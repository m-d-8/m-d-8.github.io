Projects

Project 1: flavorFeed


Built with: Swift, Firebase, Figma

flavorFeed is an IOS application created to connect like-minded people who share interest in food. This application allows users to share pictures of their meals, comment on friends meals, and tag restaurants/recipes.

Project 2: studyBuddy


Built with: Python, Flask, Figma

studyBuddy is a latform designed to help everyone complete a task they might have trouble completing. By inputting their task, they will receive a personalized broken down checklist with an extimated alloted timer running on the side.

Project 3: speedType


Built with: JavaScript, HTML, CSS

Welcome to the exhilarating speedType website, where users can put their typing skills to the test against a ticking timer. This website is built using a combination of JavaScript with a powerful library, HTML and CSS for a minimalistic design.

Project 4: showDown


Built with: C#, Unity, Photoshop

Introducing showDown, an innovative and inclusive competitive video game crafted on Unity, specifically designed to cater to the needs of the visually and physically impaired gaming community. Showdown redefines the gaming landscape by providing a thrilling, accessible gaming experience for all.
